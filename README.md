About Me

I'm Samartha GS, a student and full-stack web developer from Sagara, Karnataka, India.

I build complete web applications from the ground up — frontend, backend, database, APIs, authentication, realtime features, deployment, and UI design.

I work mainly with JavaScript, React, Next.js, Node.js, Express, MongoDB, Firebase, and Supabase. I also build projects using Socket.io, Razorpay, Cloudinary, Google OAuth, Telegram bots, and the Spotify API.

I care a lot about clean UI, smooth user experience, and building real products instead of only tutorials or demos.

Currently open to collaborations, internships, freelance work, and open-source contributions.
